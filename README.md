# eclipse-site-archives

## Prerequisites

* Eclipse
* Git-LFS
* Zip

## Instructions

1. Mirror target Artifact and Metadata repositories using Eclipse in headless mode
2. Zip mirror folder into an archive
3. Upload to eclipse-site-archives repository
